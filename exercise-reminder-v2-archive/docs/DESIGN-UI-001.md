## 一、 设计语言

### 1.1 设计关键词
- **简洁**：去除一切不必要的元素
- **清晰**：信息层次分明，一目了然
- **友好**：温暖的颜色和圆润的边角
- **高效**：大字体，易读易操作

### 1.2 设计目标
- 最小化视觉干扰
- 最大化信息可读性
- 保持一致的视觉语言

## 二、 颜色系统

### 2.1 主色调

| 颜色 | 用途 | Hex | RGB |
|------|------|-----|-----|
| **主色** | 主要按钮、强调 | `#4CAF50` | `76, 175, 80` |
| **辅助色** | 次要按钮、信息 | `#2196F3` | `33, 150, 243` |
| **成功色** | 完成反馈 | `#4CAF50` | `76, 175, 80` |

### 2.2 中性色

| 颜色 | 用途 | Hex |
|------|------|-----|
| **背景主色** | 窗口背景 | `#FFFFFF` |
| **背景次色** | 卡片背景 | `#F5F5F5` |
| **边框色** | 分隔线 | `#E0E0E0` |
| **文本主色** | 主要文字 | `#212121` |
| **文本次色** | 次要文字 | `#757575` |

### 2.3 倒计时颜色

| 时间段 | 颜色 | Hex |
|--------|------|-----|
| > 50% | 绿色 | `#4CAF50` |
| 30-50% | 黄色 | `#FFC107` |
| < 30% | 橙色 | `#FF9800` |
| < 10秒 | 红色（闪烁） | `#F44336` |

## 三、 字体系统

### 3.1 字体家族

```css

/* 等宽字体（倒计时） */
```

## 十、 统一设计规范

### 10.1 窗口行为规范

**所有提醒弹窗必须遵守**：

| 行为 | 规范 | Qt 实现 |
|------|------|---------|
| 置顶 | 强制置顶，始终在最前 | `Qt.WindowStaysOnTopHint` |
| 可拖动 | 可拖动移动窗口位置 | 自定义鼠标事件或标题栏 |
| 不可关闭 | 倒计时期间无法关闭 | 无关闭按钮，屏蔽 Esc |
| 无边框 | 站立/远眺无边框 | `Qt.FramelessWindowHint` |
| 有标题栏 | 微运动保留标题栏 | 标题栏可拖动 |

### 10.2 图标风格规范

**使用 emoji 作为图标，统一风格**：

| 用途 | 图标 | 尺寸 |
|------|------|------|
| 站立 | 🧍 | 48-64pt |
| 运动 | 🏃 | 48-64pt |
| 远眺 | 👁️ | 48-64pt |
| 设置 | ⚙️ | 24pt |
| 统计 | 📊 | 24pt |
| 主窗口 | 🏠 | 24pt |
| 托盘 | 💪 | 16pt |

### 10.3 倒计时颜色规范

**统一的时间-颜色映射**：

```
> 50% 时间 → 绿色 (#4CAF50)
30-50% 时间 → 黄色 (#FFC107)
< 30% 时间 → 橙色 (#FF9800)
< 10 秒 → 红色 + 闪烁 (#F44336)
```

### 10.4 字体规范

**统一字体使用**：

```css
/* 倒计时数字 */
font-family: "Consolas", "Monaco", monospace;
font-size: 96-120pt;

/* 标题文字 */
font-family: "Microsoft YaHei UI", "PingFang SC", sans-serif;
font-size: 48pt;
font-weight: Bold;

/* 正文 */
font-family: "Microsoft YaHei UI", "PingFang SC", sans-serif;
font-size: 16pt;
```

### 10.5 圆角规范

**统一圆角大小**：

| 组件 | 圆角 |
|------|------|
| 大弹窗 | 16px |
| 小弹窗 | 12px |
| 卡片 | 12-16px |
| 按钮 | 8px |
| 输入框 | 4-8px |
