# 灵动休息健康助手 - 使用说明

## 首次启动

1. 运行应用：
   ```
   python src/main.py
   ```

2. 首次启动会自动打开**初始设置**对话框，请设置：
   - **用户信息**：体重（用于计算热量消耗）
   - **提醒设置**：各类提醒的间隔时间
   - **音频设置**：音量和语音播报

3. 设置完成后，应用会在后台运行，系统托盘会出现 💪 图标

## 测试功能

如果想要立即测试各种提醒弹窗，运行：

```
python test_app.py
```

这会打开一个测试窗口，可以手动触发各种提醒。

## 使用演示菜单

```
python demo.py
```

提供图形化菜单，可以测试：
- 强制站立提醒
- 微运动提醒
- 强制远眺提醒
- 惩罚模式

## 系统托盘菜单

右键点击系统托盘图标可以：
- 查看统计
- 打开设置
- 暂停/恢复提醒
- 退出应用

## 默认提醒间隔

- **强制站立**：30-60 分钟随机
- **微运动**：45-75 分钟随机
- **强制远眺**：60-90 分钟随机

如需修改，请在设置中调整。

## 数据文件位置

- 数据库：`data/app.db`
- 配置文件：`data/config.json`

如需重置应用，删除这两个文件后重新运行即可。
